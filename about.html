Anarchy is a Discord API wrapper like Discord.Net or DSharpPlus. What makes Anarchy different from most other wrappers however is it's extended functionality, lightweightness and rawness. I am a firm believer that an API wrapper should be as raw / close to what's returned by the API as possible without it becoming impractical. Anarchy follows this concept with things like it's API-bound classes. Anarchy is in full support of botting on user accounts, and therefore has user-only functionality too, ranging from nitro subscriptions to livestreaming (Go Live).

The project was started by iLinked (that's me), and i've been the only developer so far.
